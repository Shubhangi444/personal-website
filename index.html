Web Design

A web designer works on the appearance, layout and content of a website. Layout refers to how information is structured and categorized. A good web design is easy to use, aesthetically pleasing, and suits the user group and brand of the website. Many webpages are designed with a focus on simplicity, so that no extraneous information and functionality that might distract or confuse users appears.

UI/UX

A UI (User Interface) deals with the application's graphical layout, which includes buttons, screen layout, animations, transitions, micro-interactions, and so on. In short, UI is all about how things look.It should be enjoyable, satisfactory, and free from frustration. The UI design should communicate the brand value of the company/organization.
